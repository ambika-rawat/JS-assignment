# JS Assignment

## Q1: Digit Gatekeeper

**Approach:** Loop from L to R, check divisibility, no zero digit, and prime digit sum. Count valid numbers.
**Time Complexity:** O(N)

---

## Q2: Roll-Seed Lock

**Approach:** Apply given transformation 3 times, then check 3-digit range and middle digit.
**Time Complexity:** O(1)

---

## Q3: Mirror Corridor

**Approach:** Try X from 0 to 100000, check if N+X is palindrome and divisible by K.
**Time Complexity:** O(N)

---

## Q4: Fare Calculator

**Approach:** Apply formula step-by-step, adjust based on conditions, round to nearest multiple of 5.
**Time Complexity:** O(1)

---

## Q5: Skipping Numbers

**Approach:** Add numbers excluding multiples of (seed+2) until sum ≥ N. Track m and sum.
**Time Complexity:** O(N)

---

## Q6: Contest Score Judge

**Approach:** Calculate score, adjust rules, then decide PASS/FAIL.
**Time Complexity:** O(1)
